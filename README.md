# AAWEB Site Credits

Elementor widget, Gutenberg block and shortcode for displaying professional website credits with the current year.

## Features

- Elementor widget
- Gutenberg block
- Shortcode support
- Classic Editor support
- Responsive frontend output
- Optional developer credit
- Settings page
- Settings shortcut on the WordPress Plugins screen

## Usage

Go to **Settings → AAWEB Site Credits** to configure default values.

Use the shortcode:

```txt
[aaweb_site_credits]
```
