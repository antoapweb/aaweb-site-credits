Professional Elementor widget, Gutenberg block and shortcode for displaying website credits.

== Description ==

AAWEB Site Credits allows you to display professional website credits using an Elementor widget, Gutenberg block or shortcode.

Perfect for web designers, developers, agencies, freelancers, WordPress websites and WooCommerce stores.

Features:

* Elementor widget
* Gutenberg block with visual styling controls
* Shortcode support
* Classic Editor support
* Responsive design
* Footer ready
* Custom text
* Custom links
* Gutenberg typography controls
* Gutenberg color controls
* Gutenberg spacing controls
* Lightweight code
* Settings shortcut on the Plugins screen

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/aaweb-site-credits` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the Plugins screen in WordPress.
3. Go to Settings → AAWEB Site Credits to configure default values.
4. Add the Elementor widget, Gutenberg block or use the `[aaweb_site_credits]` shortcode.

== Frequently Asked Questions ==

= Does it require Elementor? =

No. Elementor is optional. The shortcode and Gutenberg block can work without Elementor.

= Does it work with Gutenberg? =

Yes. You can use the AAWEB Site Credits block or the shortcode inside Gutenberg blocks.

= Does it work with WooCommerce? =

Yes. It is compatible with WooCommerce websites.

= Is it responsive? =

Yes. It works on desktop, tablet and mobile devices.
